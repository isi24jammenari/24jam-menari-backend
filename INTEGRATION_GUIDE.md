⚠️ C. Inkonsistensi Status Expired (Backend)Masalah: Di WebhookController.php, jika Midtrans mengirim status expire, BE mengubah status di database menjadi failed. Namun, di AuthController.php, BE mencoba membaca pesan error dari status expired. Akibatnya user akan melihat pesan "Pembayaran gagal" padahal aslinya "Waktu habis".✅ Tugas BE:Ubah baris di WebhookController.php:PHP// SEBELUMNYA:
$transactionStatus === 'expire' => $booking->update(['status' => 'failed']);

// UBAH MENJADI:
if ($transactionStatus === 'expire') {
    $booking->update(['status' => 'expired']); // Gunakan 'expired' secara eksplisit
}
